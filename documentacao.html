//O site Use&Reuse é uma plataforma virtual que tem como objetivo ajudar na moda sustentavel de forma simples e barata para todos. Página de Login: A página de login tem campos para inserir o e-mail e a senha. tendo codigo dos campos para garantir que não estejam vazios. Os dados de login são verificados em um objeto JSON contendo informações do usuário, como e-mail e senha. Página de Cadastro: A página de cadastro permite aos usuários criar uma nova conta na loja. Os campos obrigatórios no formulário de cadastro incluem "Nome", "E-mail" e "Senha". depois do cadastro bem-sucedido, os detalhes do novo usuário são adicionados ao objeto JSON com informações do usuário. Autenticação: login bem-sucedido, o usuário é redirecionado para a página principal da loja. A autenticação é feita verificando os dados de login no objeto JSON. Experiência do Usuário: fornecemos feedback claro em caso de erros de login ou cadastro. Mensagens de erro claras são exibidas se o login falhar, indicando que o e-mail ou a senha estão incorretos. Estilo e Design: O site segue um design simples. As páginas de login e cadastro estão em conformidade com o estilo geral da loja, utilizando cores suaves. Instruções para Execução e Teste: Baixar Arquivos: Baixe os arquivos do site (index.html, styles.css e script.js) para o seu computador. Abrir no Navegador: Abra o arquivo index.html em um navegador da web (Google Chrome, Firefox, Safari, etc.). Testar Funcionalidades: Página de Login: Insira um e-mail e senha válidos para fazer login. Caso contrário, uma mensagem de erro será exibida se os campos estiverem vazios ou se os dados estiverem incorretos. Página de Cadastro: Preencha os campos obrigatórios (nome, e-mail e senha) e clique em "Cadastrar" para adicionar um novo usuário. Após o cadastro bem-sucedido, uma mensagem de sucesso vai aparecer. Autenticação: Após o login ou cadastro bem-sucedido, uma mensagem de boas-vindas será exibida, indicando que o usuário está autenticado.
